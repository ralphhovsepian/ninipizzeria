# ninipizzeria
Nini Pizzeria is a front end template of a pizza restaurant online website.

## Features

The website is mobile responsive and you can browse through the menu, add to cart and check the total price of your selected items in the cart.

## Technologies used

- HTML5
- CSS3/Sass
- FontAwesome
- JS/ES6
- Sweet Alert
- Parcel

## How to install?

```
git clone https://github.com/ralphhovsepian/ninipizzeria.git
npm install
npm run dev
```
